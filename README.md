### E-Tourism Platform (Work in Progress)

This project is a full-stack e-tourism platform currently under development.

It includes a backend built with Node.js, Express, TypeScript, and MongoDB, and a frontend developed using React, TypeScript, and Vite. The two are connected through RESTful APIs.

The platform allows users to browse tourism offers, register and log in, and create bookings. Each offer contains details such as destination, description, price, and availability.

Authentication is handled using JWT, with password hashing for security. The system also supports role-based access, enabling administrators to manage offers through CRUD operations.

Future features include filtering and sorting offers, pagination, and an admin dashboard.
